### Expressões Condicionais

#### If

##### JavaScript

```javascript
if (a === b) {
  return 'É igual!'
}
```

##### PHP

```php
if ($a === $b) {
  return "É igual!";
}
```

#### Else

##### JavaScript

```javascript
else {
  return 'Não é igual!'
}
```

##### PHP

```php
else {
  return "Não é igual!";
}
```

#### Else if

##### JavaScript

```javascript
else if (a > b) {
  return 'É maior!';
}
```

##### PHP

```php
else if ($a > $b) {
  return "É maior!";
}

// Ou

elseif ($a > $b) {
  return "É maior!";
}
```
