### Declaração de funções

#### JavaScript

```javascript
var foo = function (a) {
  return a
}

// Ou

let foo = (a) => a

// Ou

let foo = (a) => { return a }

// Ou

function foo (a) {
  return a
}
```

##### PHP

```php
$foo = function ($a) { return $a; };

// Ou

function foo($a) {
  return $a;
}
```

##### Java

```java
public int foo(int a) { return a; } // O tipo `int` pode variar para qualquer outro tipo
```

##### Python

```python
def foo(a):
  return a
```
