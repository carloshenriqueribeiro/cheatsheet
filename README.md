### Expressões Condicionais

#### If

##### JavaScript

```javascript
if (a === b) {
  return 'É igual!';
}
```

#### Else

##### JavaScript

```javascript
else {
  return 'Não é igual!';
}
```

#### Else if

##### JavaScript

```javascript
else if (a > a) {
  return 'É maior!';
}
```
